## React Component for rendering an Admin

A basic component that renders the name, avatar, and status of an Admin

### Component usage

```js
<AdminCard name="Charles Oliver" status="Pending" />
```

### Using props to customize the text

Modify the text to see it change live:

```js live
<AdminCard name="John Doe" status="Accepted" />
```
